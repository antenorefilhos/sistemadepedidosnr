# Templates

Esta página cataloga os templates utilizados para a manutenção desta wiki:

- **Dashboard / Home**: Usado para configurar o painel central da wiki.
- **Contexto / Contexto Atual**: Estrutura contendo seções automáticas (`AUTO START`/`AUTO END`) e observações manuais (`MANUAL START`/`MANUAL END`).
- **Contexto / Onde Parei**: Ponto de retomada curto.
- **Histórico**: Registro cronológico de alterações do projeto.
- **Agentes**: Cadastro e papel das IAs ativas.
