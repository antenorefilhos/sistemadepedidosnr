```json
    "target": "es5",
    "lib": ["es5", "dom"],
    "types": ["cypress", "node"],
    "esModuleInterop": true
  },
  "include": ["**/*.ts"]
}
```

Isso instrui o editor a carregar as definições de tipo do `cypress` e do `node` (que já estão instaladas no projeto) para qualquer arquivo `.ts` dentro do diretório `cypress/`.
