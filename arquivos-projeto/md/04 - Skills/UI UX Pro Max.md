# UI/UX Pro Max

Skill de apoio para revisar e evoluir a experiencia visual do storefront e do admin com foco em clareza operacional, responsividade, consistencia de UI kit e maturidade visual.

## Uso

- Aplicar em refatoracoes de telas, componentes e fluxos de alta exposicao.
- Validar densidade, hierarquia, estados de erro/vazio/loading e comportamento mobile.
- Manter alinhamento com [[Agente Designer]] e com os componentes `ui/*`.
