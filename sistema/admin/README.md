# Admin - Antenor & Filhos

## Stack

- React 18
- Vite 4
- ApexCharts
- React Query

## Execucao com Docker

```bash
cd ../
docker compose up -d admin
```

## Execucao local

```bash
npm install
npm run dev
```

## Configuracao

- `VITE_API_URL=http://localhost:3001`
